## 1.简介
这是一个基于深度学习的高光谱解混的程序，主要是做盲解混。这里主要是想做两件事情：
- 记录一下曾经做的实验和资料，提供一个运行实验的程序代码方案
- 分享一些非常有价值的库

## 2.使用
- 通过requirements.txt安装py库，或者自行安装（建议版本至少为py37+pytorch 1.0.0）；
- 在`main.py`中写配置信息，然后点击运行即可
- [说明文档](_docs/unmixing_python_api.pyi)
